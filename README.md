# Daily Anime

Simple web application that displays currently airing anime and the days those episodes update/air

## Technologies

- HTML5
- CSS3 
- JavaScript
- jQuery
- jQuery Modal
- Google Fonts
- CORS Anywhere Proxy
- Jikan - Unofficial MyAnimeList API
